# 미션 - 점심 메뉴 추천

## 구현 순서

### 1단계 - 코치, 음식 관련 핵심 도메인 설계
- [x] 코치들 도메인 설계
  - 코치 리스트를 가지고 있는 일급 컬렉션
  - 도메인 유효성 검사
    - 2~5명
- [ ] 코치 도메인 설계
  - 값 : 이름, 못먹는 메뉴들, 선택된 메뉴들(월~금 순서), 카테고리 카운트(HashMap 사용)
  - 도메인 유효성 검사
    - 2~4글자의 이름
    - 0~2개의 못 먹는 음식
- [ ] 메뉴들 도메인 설계
  - 그냥 List<String>으로 둘 예정
- [ ] 메뉴
  - 그냥 String으로 둘 예정
- [ ] 메뉴 카테고리 판별기(enum)
  - 순회해서 메뉴가 어떤 카테고리인지 판별하는 용도
